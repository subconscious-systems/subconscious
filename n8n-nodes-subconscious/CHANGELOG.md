# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-02-03

### Added
- Initial release of n8n-nodes-subconscious
- **Run AI Agent**: Create agent runs with instructions and wait for completion
- **Engine Support**: TIM-Edge, TIM-GPT, and TIM-GPT-Heavy engines
- **Platform Tools**: 
  - Web Search (Google)
  - Webpage Understanding
  - Parallel Search
  - Parallel Extract
  - Exa Search
  - Exa Crawl
  - Exa Find Similar
- **Structured Output**: JSON Schema support for formatted responses
- **Custom Tools**: Add your own function tools via JSON configuration
- **Polling**: Automatic polling with configurable interval and timeout
- **Credentials**: Bearer token authentication with the Subconscious API
