# Convex Real-time App

AI todo assistant with real-time updates using Subconscious + Convex.

Chat with the agent to manage your todos. Changes appear instantly in the UI via WebSocket.

## Setup

### 1. Install dependencies

```bash
npm install
```

### 2. Start Convex (generates required files and environment variables)

```bash
npx convex dev
```

This creates a Convex project and generates `convex/_generated/`. It also auto-generates `.env.local` with `VITE_CONVEX_URL` and other environment variables. Keep this running in a terminal.

**Note:** Convex automatically generates `CONVEX_SITE_URL` for use in backend functions (httpActions). This is available as `process.env.CONVEX_SITE_URL` and points to your `.convex.site` URL.

### 3. Set Convex Dashboard environment variables

Go to [dashboard.convex.dev](https://dashboard.convex.dev) > Your Project > Settings > Environment Variables:

| Variable | Value |
|----------|-------|
| `SUBCONSCIOUS_API_KEY` | Your key from [subconscious.dev](https://subconscious.dev) |

### 4. Run the app (in a new terminal)

```bash
npm run dev
```

Open http://localhost:5173

## How It Works

```
Browser (React)                    Convex Backend
     |                                   |
     |-- POST /chat ------------------>  |
     |                                   |-- builds messages from history
     |                                   |-- calls Subconscious chat/completions
     |                                   |
     |                            Subconscious
     |                                   |-- returns tool_calls (if needed)
     |                                   |
     |                                   |-- executes tool via ctx.runMutation
     |                                   |-- database updates
     |<-- WebSocket update --------------|
     |                                   |
     |                                   |-- feeds result back, loops
     |                                   |-- returns final text answer
     |<-- assistant message -------------|
```

1. User sends message to `/chat` endpoint
2. Convex loads conversation history and current todos
3. Convex calls Subconscious `POST /v1/chat/completions` (OpenAI-compatible)
4. If the model requests tools, Convex executes them directly via `ctx.runMutation`
5. Results are fed back into the message thread; the loop repeats until the model returns a final answer
6. Database mutations trigger WebSocket updates to React — the UI can update before the AI finishes

## Project Structure

```
src/
  main.tsx              # Entry point, Convex setup
  App.tsx               # Layout with Chat + TodoList
  components/
    Chat.tsx            # Chat interface
    TodoList.tsx        # Real-time todo display

convex/
  schema.ts             # Database tables
  todos.ts              # Todo mutations
  messages.ts           # Chat history
  tools.ts              # HTTP endpoints (unused by Subconscious, kept for reference)
  chat.ts               # Calls Subconscious API + runs tool loop
  http.ts               # Routes HTTP endpoints
  _generated/           # Auto-generated by `npx convex dev`
```

## Adding Tools

Add a mutation in `convex/todos.ts`:

```typescript
export const setPriority = mutation({
  args: { id: v.id("todos"), priority: v.string() },
  handler: async (ctx, { id, priority }) => {
    await ctx.db.patch(id, { priority });
    return { success: true };
  },
});
```

Add the OpenAI-format tool definition in `convex/chat.ts` (in the `TOOLS` array):

```typescript
{
  type: "function",
  function: {
    name: "setPriority",
    description: "Set the priority of a todo item",
    parameters: {
      type: "object",
      properties: {
        id: { type: "string", description: "The todo ID" },
        priority: { type: "string", description: "low, medium, or high" },
      },
      required: ["id", "priority"],
    },
  },
}
```

Add a case in the `executeTool` switch in `convex/chat.ts`:

```typescript
case "setPriority": {
  const id = typeof args.id === "string" ? args.id : "";
  const priority = typeof args.priority === "string" ? args.priority : "";
  if (!id || !priority) return { error: "Missing required parameters" };
  return await ctx.runMutation(api.todos.setPriority, {
    id: id as Id<"todos">,
    priority,
  });
}
```

## Troubleshooting

### "Failed to resolve import convex/_generated/api"

Run `npx convex dev` first. This generates the required files.

### Tools not being called

Make sure `SUBCONSCIOUS_API_KEY` is set in the Convex Dashboard and the model has instructions about when to use tools. The tool loop runs inside the `httpAction` — no external callback URLs are needed.
