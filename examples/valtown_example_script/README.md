# Val Town Example (Subconscious)

Learn how to use the Subconscious API with automation scripts on Val Town.

## Quick start

1. Open the getting-started Val Town repo:  
   [subconscious-getting-started](https://www.val.town/x/subconscious_ajaya/subconscious-getting-started)
2. Remix the repo to your own Val Town account.
3. Add environment variable in Val Town:
   - `SUBCONSCIOUS_API_KEY=your_key`
4. Run the Val you want to test.

## Included example Vals

- `01_script.ts` - one-shot agent query
- `02_http.ts` - HTTP endpoint backed by an agent
- `03_cron.ts` - scheduled agent task
- `04_email.ts` - email-triggered agent workflow

## Required env vars

- `SUBCONSCIOUS_API_KEY` (required)
